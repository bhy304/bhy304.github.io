접속한 회원 세션 관리, 헤더 메뉴 수정

--- 

# 세션(session)이란?
쿠키와 마찬가지로 서버와의 관계를 유지하기 위한 수단이다.<br>
단, 쿠키와 달리 클라이언트의 특정 위치에 저장되는 것이 아니라, **서버상에 객체로 존재**한다.
따라서 <u>세션은 서버에서만 접근이 가능하여 보안이 좋고, 저장할 수 있는 데이터에 한계가 없다.</u>

---

# 회원 세션 관리

##### 1. 로그인 성공시 사용자의 아이디를 세션에 저장
: ```loginAction.jsp```

  ```java
  session.setAttribute("userID", dto.getUserID()); //세션에 데이터 저장
  ```

##### 2. 로그아웃 기능 추가<br>
: ```logoutAction.jsp```

```java
<%
session.invalidate(); //세션의 모든 데이터를 삭제
response.sendRedirect("main.jsp"); //로그아웃이 되면 main 페이지로 이동
%> 
```

##### 3. ```header.jsp``` 수정
헤더 메뉴가 로그인 상태에 따라 바뀌어야한다. <br>
세션에 저장되어 있는 회원의 아이디가 null값이면 회원가입과 로그인을 헤더 메뉴에 표시하고,
null값이 아니라면 회원정보수정과 로그아웃을 표시한다. 

```java
<c:set var="path" value="${pageContext.request.contextPath}"></c:set> 
//웹어플리케이션의 contextPath를 얻음
<header id="header">
  <h1 class="logo"><a href="${path}/main.jsp">메인</a></h1>
  <nav class="nav">
  <ul>
    <li><a href="${path}/#">게시판</a></li>
<c:if test="${sessionScope.userID == null }"> // 로그인 X
    <li><a href="${path}/join.jsp">회원가입</a></li>
    <li><a href="${path}/login.jsp">로그인</a></li>
</c:if>
<c:if test="${sessionScope.userID != null }"> // 로그인 O
    <li><a href="${path}/#">회원정보수정</a></li>
    <li><a href="${path}/logoutAction.jsp">로그아웃</a></li>
</c:if>
  </ul>
  </nav>
</header>
```
